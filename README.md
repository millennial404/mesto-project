**О проекте**

Проект «Mesto» - интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.
Проектная работа создана при обучении на курсе Web+ Яндекс практикум.
Адаптивная верстка, HTML, CSS, JS.
